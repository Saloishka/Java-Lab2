Дослідити список із п.4: Створити клас для статистичних даних, та зібрати у нього (за допомогою свого Collector) елементи зі списку. Вивести  на екран наступні значення: мінімальне і максимальне значення поля Г, середнє значення поля Г та стандартне відхилення. 

Дослідити значення поля Г в отриманому списку: визначити  міжквартильний розмах та ідентифікувати викиди в отриманому списку. Згрупувати список по належності значення поля Г до викидів та підрахувати кількість значень у кожній категорії. В результаті має бути така структура даних: {“data”: 412, “outliers”: 88}
